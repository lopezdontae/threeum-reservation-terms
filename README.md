# Threeum Cella Computer Reservation Terms of Service

Standalone public-facing Terms of Service for the Cella computer reservation fee, hosted via GitHub Pages.

**Live URL:** https://lopezdontae.github.io/threeum-reservation-terms/

This repository is intentionally independent of the main Threeum website. It exists only to provide a stable, publicly accessible URL that can be linked from Stripe checkout and any other reservation surfaces.

## Updating

Edit `index.html` and commit. GitHub Pages rebuilds automatically. Increment the version and effective date in the header when making material changes.
